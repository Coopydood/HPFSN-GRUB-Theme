## Variants

Even while maintaining the same shared fluent design language, each variant of  HyperFluent has a distinct look and feel, tailored around the subject of the theme.

**OS / Distros**
- Arch Linux 
- Ubuntu 
- Redhat 
- Fedora 
- NixOS 
- endeavourOS 
- Debian 
- Linux Mint 
- Manjaro 
- OpenSUSE 
- OpenSUSE (Tumbleweed) 
- macOS 
- Windows (Dark) 
- Windows (Light) 
- Gentoo
- Alt Linux

**Other**
- Generic
